# 📈 Sales & Demand Forecasting using Machine Learning

## 📌 Overview

This project predicts future weekly sales using Machine Learning techniques. It helps businesses forecast demand, optimize inventory, and improve decision-making.

The project uses historical sales data from Walmart stores and applies regression algorithms to estimate future sales trends.

---

## 🚀 Features

- Data Cleaning
- Feature Engineering
- Weekly Sales Forecasting
- Linear Regression Model
- Random Forest Regressor
- Model Evaluation
- Sales Trend Visualization
- Actual vs Predicted Comparison
- Feature Importance Analysis
- Future Sales Prediction
- Business Insights
